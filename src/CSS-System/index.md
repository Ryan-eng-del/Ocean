# Style System

这是 Ocean UI 的第一大系统 Style System，从左侧栏可以看到 有各种各样的名称，这些都是 Ocean UI 提供的 CSS 单位，你可以在对应的 CSS 属性上使用单位，基本上涵盖了所有常见的单位。

## Colors

你可以这样使用,在任何需要颜色,或者需要覆盖颜色的地方，你都可以使用这些值

```javascript
<ocean.div bg={"slate.300" c="blue.500"}/>
```
