# Radio

## 何时使用

- 用于在多个备选项中选中单个状态。
- 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

## 基本使用

**Ocean UI** 导出了 **Radio** 以及 **RadioGroup** 组件。

**exam**: 提供考试模式，也就是选项包括在组件当中，非考试模式则是在组件之外。
**solid**: 提供实心模式，也就是颜色直接填充组件，非实心是边框修饰组件。
**type**: 提供不同的类别，也就是按钮的类别。
