# Tab

选项卡切换组件。

## 何时使用

- 提供平级的区域将大块内容进行收纳和展现，保持界面整洁。
- 卡片式的页签，提供可关闭的样式，常用于容器顶部。
- 既可用于容器顶部，也可用于容器内部，是最通用的 Tabs。
