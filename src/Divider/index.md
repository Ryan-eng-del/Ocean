# Divider

分隔内容的文档分割线。

### 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

### 基本使用

`Ocean UI` export 3 个与 `Divider` 有关的组件。可以对这三个组件，进行自由的组合，也可以由 `Ocean Style System` 进行任何样式覆盖和修改。

- **Divider**：Divider 组件的 Wrapper。
- **DividerLine**: Divider 组件的分割线。
- **DividerText**: Divider 组件的分割内容。
