# Message

全局展示操作反馈信息。

## 何时使用

- 可提供成功、警告和错误等反馈信息。
- 顶部居中显示并自动消失，是一种不打断用户操作的轻量级提示方式。

## 基本使用 - Usage

`Ocean UI` export 出 useMessage Hook, 用来对弹窗进行管理

`const { message, close, update } = useMessage();`

- **message**：创建弹窗。
- **close**: 关闭弹窗，传入弹窗 Id，进行关闭。
- **update**: 更新弹窗，传入弹窗 Id，进行更新。
